USAGE:
    sane-fmt [OPTIONS] [FILES]...

ARGS:
    <FILES>...
            Files to process

            If none are provided, a default set of files will be assumed

OPTIONS:
        --color <COLOR>
            When to use terminal color

            [default: auto]
            [possible values: auto, never, always]

        --details <DETAILS>
            File diff detail

            [default: name]
            [possible values: count, name, diff]

    -h, --help
            Print help information

        --hide-passed
            Do not log passed filenames

    -I, --include <INCLUDE>
            Files whose contents contain paths to target files (`-` means stdin, other strings mean
            text file)

        --log-format <LOG_FORMAT>
            Format of log messages

            [default: human]
            [possible values: human, github-actions]

        --stdio
            Reads unformatted code from standard input, prints formatted code to standard output,
            then exits

    -V, --version
            Print version information

    -w, --write
            Whether to write or check
